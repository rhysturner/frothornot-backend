# frothornot-backend

This is a Node app for the backend services of my surf report app.

### Features
1. get surfreport based on location
1. user profile
1. predict conditions
1. machine learning based on your preferences
 

 ### Dev

To run the app

`$ cd frothornot-backend`
`$ npm i`
`$ npm start`
